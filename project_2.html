    • 3D Dynamic Robotic Arm Modeled in Matlab

    My introduction to MATLAB occurred during my engineering studies at UBC. While the curriculum covered basic applications of the software, I quickly recognized its potential for more sophisticated endeavors.

    One of my initial projects involved modeling a robotic arm. Beginning with a straightforward 2D multi-link arm manipulator, I gradually expanded the project into the realm of 3D, discovering new challenges and possibilities along the way.

    Part 1: Beginning in 2D

    Starting with a 2D simulation that I had created for my BMEG 425 course, I wanted to adapt the manipulability ellipses and force ellipses to my 3D simulation. Another feature I wanted to adapt from my old code was the crash detection and correction when the manipulator interacts with the physical boundary imposed on it.

    A feature I added to the 3D manipulator that was not included in the 2D simulation was a smooth trajectory algorithm. I decided on implementing a cubic polynomial trajectory algorithm which can be found on the github repository. For simplicity, the algorithm doest involve angular jacobian calculations and that was instead handled by a simple orientation selection input from the user that controls the spherical joint connected to the end effector.
